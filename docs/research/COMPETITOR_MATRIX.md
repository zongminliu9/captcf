build one coherent, original product. We copied **no** logos, trademarks, brand names,
marketing copy, testimonials, protected images, or proprietary item banks.

## How CapTCF differentiates

- A **single transparent recommendation** with a machine-readable reason code — not a wall
  of equal-weight cards.
- Mistakes and vocabulary flow into an **SM-2 spaced-repetition** queue automatically.
- **Guest-first**: full practice + analysis with no signup; data merges losslessly on register.
- **Runs with zero paid APIs**: real local audio, deterministic writing/speaking feedback,
  entitlement simulator.
- Every estimate is explicitly **unofficial** and tied to the current exam structure version.

## Marketing honesty guardrails (enforced in the product)

No fake user counts, pass rates, testimonials, official partnerships, discount countdowns,
or "guaranteed NCLC" claims. Pricing/FAQ/legal pages state independence and unofficial status.
